Professional GSAP animation toolkit for WordPress and Elementor.

== Description ==

This plugin provides a clean OOP architecture for future GSAP and Elementor integration.

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/gsap-elementor-toolkit directory.
2. Activate the plugin through the WordPress admin panel.
3. Start extending the plugin structure in the includes/ folders.
